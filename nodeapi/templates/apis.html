Welcome to Tree Plantation API

API Overview

This API allows you to interact with tree plantation data, including uploading node data, retrieving current values, and accessing detailed node information.

Available Endpoints

1. Home Endpoint

URL: {{domainUrl}}/

Method: GET

Description: Displays the total records and current values of each unique node.

2. Upload Data

URL: {{domainUrl}}/uploaddata

Method: GET

Parameters:

  • nodename: The name of the node.
  • height: The height of the node.

Description: Inserts node data along with the IP address and initializes empty latitude and longitude for new nodes.

Example: {{domainUrl}}/uploaddata?nodename=Tree1&height=10

3. Display Node Data

URL: {{domainUrl}}/displaynode

Method: GET

Description: Displays all nodes and their details with a periodic refresh.

4. Node Details

URL: {{domainUrl}}/displaynode/<nodename>

Method: GET

Description: Displays all records for a specific node.

Example: {{domainUrl}}/displaynode/Tree1

5. JSON Node Data

URL: {{domainUrl}}/getnodejson/<nodename>

Method: GET

Description: Returns all node data in JSON format.

Example: {{domainUrl}}/getnodejson/Tree1

6. Latest Node JSON

URL: {{domainUrl}}/getnodelatestjson/<nodename>

Method: GET

Description: Returns the latest node record in JSON format.

Example: {{domainUrl}}/getnodelatestjson/Tree1

7. Edit Node Data

URL: {{domainUrl}}/edit/<nodename>

Method: GET/POST

Description: Displays a form to edit latitude and longitude for the specified node.

Example: {{domainUrl}}/edit/Tree1

8. Map Visualization

URL: {{domainUrl}}/mapit

Method: GET

Description: Generates a map showing the locations of all nodes using Folium.

9. Display Map Visualization on your website using iframe tag

URL: {{domainUrl}}/getmap

Method: GET

Description: Generates a map showing the locations of all nodes using Folium.

How to Use

You can use these endpoints by making GET requests through your web browser or using tools like Postman. Ensure to provide the required parameters where necessary.
